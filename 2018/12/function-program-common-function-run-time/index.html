常用的高阶函数之控制执行次数

2018-12-04
字数: 342 阅读时长: 2mins 阅读次数: 次
分类： js
标签： js 高阶函数

控制次数其实主要从两方面着手，一方面是简单的限制函数的执行次数，而另一方面可以缓存函数的执行结果。

限制函数的执行次数

对于有些场景下可能需要控制执行次数，比如设置错误重试次数5次，比如表单提交函数只允许执行一次防止重复提交。

缓存函数的执行结果

对于一些执行成本非常高的操作或者会被反复调用的操作，可以通过缓存，限制其执行次数。
